== Description ==
Simple customizable button for direct link to user or telegram channel.

== Installation ==
1. Install plugin
2. Activate plugin
3. Go to telagram contact in the administration menu
4. Configure
5. done

== Frequently Asked Questions ==
What do I need to make the button work?
You only need to put your username or channel in the corresponding field of the telegram contact administration menu
